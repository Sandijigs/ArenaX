# ArenaX

ArenaX is a competitive gaming platform built on blockchain technology, featuring tournament management, player matchmaking, and reward distribution through smart contracts.

## Project Structure

- `backend/` - Rust-based API server with tournament and matchmaking services
- `contracts/` - Stellar smart contracts for token management, escrow, and tournament logic
- `frontend/` - Next.js React application for the gaming platform
- `.github/` - CI/CD workflows and GitHub configuration
